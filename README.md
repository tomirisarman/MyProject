Моя backend-реализация <b>образовательной платформы.</b>
Три гарда:
<ul>
    <li>Пользователь (студент)</li>
    <li>Преподаватель</li>
    <li>Администратор</li>
</ul>

Также имеется три модели:
<ul>
    <li>Course - курсы</li>
    <li>Lesson - уроки, входящие в каждый курс</li>
    <li>Homework - домашние задания, загруженные пользователями по какому-либо уроку</li>
</ul>

Разработана система публикации и скачивания материалов, отправки домашних заданий пользователем и оценки их выполнения преподавателем. Файлы сохраняются непосредственно на сервере.
